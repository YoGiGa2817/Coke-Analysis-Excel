# Coca-Cola Sales & Financial Performance Analysis

## Project Overview
This project analyzes Coca-Cola’s historical data to identify growth trends and market volatility. The analysis focuses on transforming raw transactional data into actionable business insights using advanced Excel techniques.

## Key Features

Data Cleaning: Standardized date formats and currency headers in the Coca Cola.xlsx dataset.

Custom Metrics: Calculated Daily Price Change (Close - Open) to monitor market fluctuations.

Pivot Analysis: Aggregated data to find the Average Closing Price ($55.27) and identified 2022 as the peak performance year ($61.68).

Interactive Dashboard: Built a dynamic view with Slicers and Combo Charts to visualize the relationship between Price and Trading Volume.

## Conclusion
The analysis confirms Coca-Cola’s resilience as a stable asset, showing a steady recovery and upward trajectory post-2020.
